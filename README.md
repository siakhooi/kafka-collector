# kafka-collector

collect kafka messages from multiple topics

## Installation
```
pip install kafka_collector
```
## Usage
```
$ kafka-collector -h
usage: kafka-collector [-h] [-v]

collect kafka messages from multiple topics

options:
  -h, --help     show this help message and exit
  -v, --version  show program's version number and exit

```
